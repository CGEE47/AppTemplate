# AppTemplate

### A small handy template for a flatpak style app, configured with an excecutable for both Windows and Linux.

___

#### Jobs
```
1. Resources folder/subsections DONE
2. Import PythonVers into Resources for Win/Lin/Mac DONE (No Mac)
3. .exe for Win launching main.py with flatpak Win Py DONE
4. .desktop for Lin launching main.py with flatpak Lin Py DONE
5. .app for Win launching main.py with flatpak Mac Py (Might not implement)
```
